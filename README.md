# 🎬 TECHATHON — A Different Creative Approach

An experimental and cinematic creative agency landing page built using **HTML5, CSS3, and Vanilla JavaScript**.

Designed with a focus on immersive visuals, motion-driven storytelling, and premium futuristic UI experiences.

---

## ✨ Project Highlights

- 🎥 Motion-driven storytelling
- 🌌 Immersive visual experience
- 🖱 Interactive UI systems
- 🪼 Organic Canvas animations
- 🚀 Premium futuristic aesthetics
