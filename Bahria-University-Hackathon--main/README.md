# TECHATHON — A Different Creative Approach

An experimental and cinematic creative agency landing page built using pure HTML, CSS, and JavaScript.

This project focuses on:
- Motion-driven storytelling
- Immersive visuals
- Interactive UI systems
- Organic canvas animations
- Premium futuristic aesthetics

---

# ✨ Features

## 🎨 Modern Visual Design
- Futuristic UI
- Grain texture overlay
- Neon gradient effects
- Glassmorphism-inspired elements
- Experimental typography system

## 🖱 Custom Cursor
- Animated dual-layer cursor
- Smooth interpolation movement
- Blend-mode interaction effects

## 🪼 Interactive Jelly Blob
- Canvas-powered liquid blob
- Rainbow animated gradients
- Mouse-reactive morphing
- Scroll-driven movement animation

## 📜 Scroll Interactions
- Reveal animations
- Sticky sections
- Dynamic navbar blur effect
- Smooth scrolling behavior

## 🧩 Dynamic Components
- Hero section
- Trusted brands grid
- About section
- Interactive project cards
- Awards section
- Premium animated footer

## 📱 Fully Responsive
Optimized for:
- Desktop
- Tablets
- Mobile devices
